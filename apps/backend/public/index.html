Sayr.io — Open Project Management

A modern platform for transparent collaboration. Public and private fields, role-based permissions, and a single timeline for public/internal threads.

Key features

  • Granular visibility: toggle public/private per field, label, or comment
  • Role-based permissions: separate external submitters from org members
  • Dual discussions: unified timeline with public and internal comments
  • Labels, votes, and metadata with visibility controls
  • Open API and OAuth-ready authentication
