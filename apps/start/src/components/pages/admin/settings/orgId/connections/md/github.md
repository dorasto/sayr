Sayr integrates with GitHub to keep your tasks and issues in sync. Connect one or more GitHub organizations or personal accounts, choose which repositories to sync, and link activity across both platforms.

- **Two-way task syncing** — Sayr tasks create GitHub issues and vice versa. Filter syncing by category to control which tasks flow to which repositories.
- **Reference tasks from GitHub** — Use keywords like `Fixes #42`, `Ref #18`, or `Closes #7` in commit messages, PR titles, and issue bodies to link GitHub activity to Sayr task timelines.
- **Automatic status updates** — Merging a linked pull request marks the Sayr task as Done. Keywords like `Fixes` and `Resolves` also close tasks automatically.
- **Comment syncing** — Comments on linked GitHub issues and pull requests appear on the corresponding Sayr task.
- **Pull request tracking** — Linked PRs show their full lifecycle on the task timeline: opened, synchronized, merged, or closed.
