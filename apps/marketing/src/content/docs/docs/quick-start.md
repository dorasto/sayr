This guide will help you get started with Sayr in just a few minutes.

## 1. Create an Account

Sign up at [admin.sayr.io](https://admin.sayr.io) using your GitHub account.

## 2. Create an Organization

Organizations are the core unit in Sayr. Each organization gets its own subdomain (e.g., `yourorg.sayr.io`).

1. Click "Create Organization"
2. Enter your organization name and slug
3. Choose your visibility defaults

## 3. Create Your First Task

Tasks in Sayr support:

- **Labels**: Categorize with color-coded labels
- **Priorities**: None, Low, Medium, High, Urgent
- **Statuses**: Backlog, Todo, In Progress, Done, Canceled
- **Assignees**: Assign to one or more team members
- **Rich Descriptions**: Use the block editor for formatted content

## 4. Invite Your Team

Navigate to Organization Settings > Members to invite collaborators.
